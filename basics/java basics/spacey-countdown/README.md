# Обратный отсчет SpaceY

## Задача
Обратный отсчет до запуска ракеты. Счет начинается с 70 и идет с шагом 7.

## Что использовал
- Цикл `for`
- Переменная `int`
- Шаг итерации `i -= 7`

## Что понял
Цикл `for` удобен когда заранее известно количество итераций.
Шаг может быть любым - не обязательно 1.

# Конвертер валют

## Задача
Программа конвертирует рубли в доллары, евро или юани по фиксированному курсу.

## Что использовал
- Переменные `double` и `int`
- Ввод данных через `NumberReader`
- Условия `if / else if / else`

## Что улучшил
Убрал дублирование кода — вместо трёх одинаковых блоков проверки сделал одну общую.

## Курсы валют
- USD: 81.9
- EUR: 87.7
- CNY: 11.49

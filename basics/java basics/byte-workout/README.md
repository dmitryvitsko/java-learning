# Тренировка Байта

## Задача
Программа для тренировки - 5 кругов, чередование отжиманий и пресса, бег в конце каждого круга.

## Что использовал
- Цикл `for`
- Условие `i % 2 == 0` для чередования четных и нечетных кругов
- Оператор `%` (остаток от деления)

## Что понял
Оператор % удобен для чередования - если остаток от деления на 2 равен 0, число четное.

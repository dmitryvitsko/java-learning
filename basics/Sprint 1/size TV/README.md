# Конвертер дюймов в сантиметры

## Задача
Программа переводит размер диагонали телевизора из дюймов в сантиметры.

## Что использовал
- Класс `Scanner` для считывания ввода пользователя
- Переменная `double` для дробных чисел

## Что понял
`Scanner` - основной инструмент для чтения данных с клавиатуры в Java.
`scanner.nextDouble()` считывает дробное число введенное пользователем.

# 💰 Конвертер рублей разных эпох

## 📋 Задача
Программа конвертирует современные рубли в рубли 1913 или 1984 года
и показывает сколько товаров той эпохи можно купить на эту сумму.

---

## 🧠 Теория

### Оператор switch
Альтернатива длинным if/else if для проверки одной переменной:
```java
switch (year) {
    case 1913:
        koef = 1.0 / 884;
        break;
    case 1984:
        koef = 1.0 / 337;
        break;
}
```
`break` обязателен - без него выполнение "провалится" в следующий case.

### Приведение типов
```java
int quantity = (int) (amountByYear / price);
```
`(int)` обрезает дробную часть - 3.9 становится 3.
Используется когда нужно целое число из дробного результата.

### Коэффициент через 1.0
```java
double koef = 1.0 / 884; // правильно - результат дробный
double koef = 1 / 884;   // неправильно - целочисленное деление, результат 0
```
Если хотя бы один операнд `double` - результат тоже `double`.

### switch для строк
```java
switch (item) {
    case "Корова":
        price = 60.0;
        break;
}
```
В Java switch работает не только с числами но и со строками.

---

## 🛠 Что использовал
- `switch` для чисел и строк
- Приведение типов `(int)`
- Коэффициент через `1.0` для дробного деления

---

## 💡 Вывод
switch удобнее if/else когда проверяем одну переменную на много значений.
Код становится чище и читаемее.

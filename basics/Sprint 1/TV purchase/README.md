# Калькулятор телевизора

## Задача
Программа считывает марку, диагональ и цену телевизора,
переводит дюймы в сантиметры и считает итоговую цену со скидкой.

## Что использовал
- `scanner.nextLine()` для считывания строки
- `scanner.nextDouble()` для дробного числа
- `scanner.nextInt()` для целого числа

## Что понял
Scanner умеет читать разные типы данных - строки, дробные и целые числа.
Для каждого типа свой метод: nextLine, nextDouble, nextInt.

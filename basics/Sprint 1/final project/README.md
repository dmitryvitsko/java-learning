# 🛒 Динамический список покупок на Java

## 📋 Описание

Консольное приложение для ведения списка покупок с автоматическим увеличением размера массива.

В отличие от обычного массива фиксированной длины, программа умеет самостоятельно расширять внутреннее хранилище, когда в нём заканчивается место. Пользователь продолжает работать со списком и даже не замечает, что внутри был создан новый массив.

---

## 🧠 Что реализовано

* ➕ Добавление новых товаров
* 📄 Просмотр текущего списка покупок
* 🗑️ Полная очистка списка
* 🚪 Завершение работы программы
* 🔄 Автоматическое увеличение размера массива при заполнении

---

## ⚙️ Как работает динамическое расширение

Изначально список рассчитан на 8 товаров.

```java
int currentListSize = 8;
String[] shoppingList = new String[currentListSize];
```

Когда свободных ячеек больше не остаётся, программа:

1. Создаёт новый массив в два раза больше.
2. Копирует в него все существующие элементы.
3. Переключается на работу с новым массивом.

```java
if (productCount == currentListSize) {
    int newListSize = currentListSize * 2;
    String[] newShoppingList = new String[newListSize];

    for (int i = 0; i < shoppingList.length; i++) {
        newShoppingList[i] = shoppingList[i];
    }

    shoppingList = newShoppingList;
    currentListSize = newShoppingList.length;
}
```

Таким образом приложение может хранить сколько угодно товаров, постепенно увеличивая объём памяти по мере необходимости.

## 💡 Главный инсайт

Обычные массивы в Java имеют фиксированный размер и не могут увеличиваться после создания.

Чтобы реализовать динамический список, необходимо самостоятельно создавать новый массив большего размера и переносить в него существующие данные.

Именно на этом принципе основана работа многих популярных структур данных, например `ArrayList`, который делает всё это автоматически "под капотом".

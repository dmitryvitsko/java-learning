# 🔍 Отладка массива расходов

## 📋 Задача
Найти и исправить ошибки в программе которая заполняет массив расходов за неделю.

---

## 🧠 Теория

### Ошибки которые были в коде

**1. Неправильное объявление массива**
```java
// Было - ошибка:
double expenses = double[7];

// Стало - правильно:
double[] expenses = new double[7];
```

**2. Индекс цикла начинался с -1**
```java
// Было - выход за границу массива:
for (int i = -1; i <= expenses.length; i+-)

// Стало - правильно:
for (int i = 1; i < expenses.length; i++)
```
Индексы массива от 0 до length-1. При -1 или length получим `ArrayIndexOutOfBoundsException`.

**3. Неправильный оператор**
```java
i+- // так не существует
i++ // правильный инкремент
```

**4. Логика заполнения**
```java
// Было - брали незаполненный элемент:
expenses[i] = expenses[i] + 100; // expenses[i] = 0, всегда получалось 100

// Стало - берем предыдущий элемент:
expenses[i] = expenses[i - 1] + 100; // каждый день на 100 больше предыдущего
```

---

## 🛠 Что использовал
- Массив `double[]`
- Обращение к предыдущему элементу `expenses[i - 1]`

---

## 💡 Вывод
Отладка кода - важный навык. Нужно проверять:
- правильность объявления переменных
- границы цикла - не выходить за 0 и length-1
- логику обращения к элементам массива

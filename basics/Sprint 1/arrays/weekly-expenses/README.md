# 💸 Траты за неделю

## 📋 Задача
Программа заполняет массив случайными тратами за каждый день недели,
выводит их и считает сумму за неделю.

---

## 🧠 Теория

### Создание пустого массива
```java
int[] expenses = new int[7]; // массив из 7 элементов, все = 0
```
В отличие от `String[] arr = {"a", "b"}` - здесь размер задается числом,
элементы заполняются потом.

### Генератор случайных чисел Random
```java
Random random = new Random();
random.nextInt(1000); // случайное число от 0 до 999
```
`nextInt(n)` возвращает случайное число от 0 до n-1.

### Накопление суммы в цикле
```java
int sum = 0;
for (int i = 0; i < expenses.length; i++) {
    sum += expenses[i]; // sum = sum + expenses[i]
}
```
Паттерн аккумулятора - объявляем переменную до цикла, накапливаем внутри.

---

## 🛠 Что использовал
- Массив `int[]` с заданным размером
- `Random` для генерации случайных чисел
- Три цикла `for` - заполнение, вывод, подсчет суммы

---

## 💡 Вывод
Три отдельных цикла вместо одного - каждый делает свою задачу.
Это лучше чем смешивать логику в одном цикле.

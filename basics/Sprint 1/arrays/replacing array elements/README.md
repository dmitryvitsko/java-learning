# 🍽 Перестановка блюд

## 📋 Задача
Программа меняет местами два блюда в массиве по индексам введенным пользователем.

---

## 🧠 Теория

### Массивы в Java
\```java
String[] dishes = {"Ризотто", "Тартар", "Шурпа"};
// индексы:           0          1         2
\```
Массив - набор элементов одного типа. Индексация начинается с 0.

### Обмен элементов местами

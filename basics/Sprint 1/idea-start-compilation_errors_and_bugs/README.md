# 📝 Консольный менеджер задач (Task Manager)

## 📋 Описание

Небольшое консольное приложение для управления списком задач.

Пользователь может:

* ➕ Добавлять новые задачи;
* ❌ Удалять существующие задачи;
* 📄 Просматривать весь список;
* 🚪 Завершать работу программы.

Все задачи хранятся в обычном массиве `String[]`, а взаимодействие происходит через текстовое меню в консоли.

---

## 🧠 Что изучил и применил

### Работа с массивами

Для хранения задач используется массив фиксированного размера.

```java
private final String[] tasks;
```

Максимальное количество задач задаётся константой:

```java
private static final int MAX_TASKS = 10;
```

---

### Классы и объекты

Программа разделена на два класса:

* `MainMenu` — отвечает за взаимодействие с пользователем;
* `TaskManager` — содержит всю логику работы с задачами.

Такое разделение делает код более понятным и удобным для дальнейшего развития.

---

### Цикл `while`

Главное меню работает в бесконечном цикле и завершается только после выбора пункта выхода.

```java
while (true) {
    // отображение меню
}
```

---

### Условные конструкции `if / else`

В зависимости от выбора пользователя выполняется нужное действие:

```java
if (choice == 1) {
    // добавить задачу
} else if (choice == 2) {
    // удалить
} else if (choice == 3) {
    // показать список
}
```

---

### Работа со `Scanner`

Для ввода данных используется класс `Scanner`.

```java
Scanner scanner = new Scanner(System.in);
```

Позволяет получать как числа, так и строки с клавиатуры.

---

### Добавление элемента

Новая задача записывается в первую свободную ячейку массива.

```java
tasks[taskCount] = task;
taskCount++;
```

---

### Удаление элемента

После удаления все последующие элементы сдвигаются влево, чтобы не оставалось пустых мест.

```java
for (int i = index; i < taskCount - 1; i++) {
    tasks[i] = tasks[i + 1];
}
```

После этого последний элемент очищается:

```java
tasks[taskCount - 1] = null;
taskCount--;
```

---

## 🛠 Использованные технологии

* Java
* Классы и объекты
* Массивы (`String[]`)
* Цикл `while`
* Условные операторы `if / else`
* Константы (`static final`)
* Работа с `Scanner`
* Инкапсуляция логики в отдельном классе

---

## 💡 Что можно улучшить

В будущем приложение можно сделать значительно удобнее:

* использовать `ArrayList` вместо массива фиксированного размера;
* добавить возможность редактирования задач;
* сохранять список в файл;
* отмечать задачи как выполненные;
* сортировать и искать задачи;
* реализовать графический интерфейс или REST API.

## 🚀 Итог

Это первая версия собственного консольного менеджера задач.

Проект помог закрепить работу с массивами, циклами, условиями, методами и разделением программы на несколько классов. Несмотря на простоту, именно такие небольшие приложения постепенно формируют понимание того, как строятся более крупные проекты.

# 🎲 Угадай число

## 📋 Задача

Разработать консольную игру «Угадай число».

Компьютер случайным образом загадывает число от **0 до 1000**, а игрок должен его угадать.

После каждой попытки программа подсказывает:

* **Больше** — если загаданное число больше введённого.
* **Меньше** — если загаданное число меньше введённого.
* Поздравление — если число угадано.

Игра продолжается до тех пор, пока пользователь не угадает число.

---

## 🧠 Что использовал

* Цикл `while`
* Условные операторы `if / else if / else`
* Класс `Scanner`
* Класс `Random`
* Сравнение чисел

---

## 🎲 Генерация случайного числа

Для создания случайного числа используется класс `Random`:

```java
int randomInt = new Random().nextInt(1000);
```

Метод `nextInt(1000)` генерирует число от `0` до `999`.

---

## 🔄 Основная логика игры

Игра работает до тех пор, пока пользователь не угадает число:

```java
while (userInput != randomInt)
```

На каждой итерации программа получает число от пользователя:

```java
userInput = scanner.nextInt();
```

После этого происходит сравнение введённого числа с загаданным.

---

## 🔍 Проверка вариантов

Если введённое число меньше загаданного:

```java
if (userInput < randomInt)
```

Выводится подсказка:

```text
Больше
```

Если введённое число больше загаданного:

```java
else if (userInput > randomInt)
```

Выводится подсказка:

```text
Меньше
```

Если числа совпали:

```java
else
```

Игра завершается сообщением об успехе.

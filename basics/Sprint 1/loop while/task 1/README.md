# 💰 Миллион на накопительном счёте

## 📋 Задача

Друг накопил некоторую сумму денег и положил её на накопительный счёт под **5% годовых**.

Необходимо определить, через сколько лет сумма на счёте достигнет **1 000 000 рублей**.

Начальный баланс вводится пользователем через консоль.

---

## 🧠 Что использовал

* Цикл `while`
* Класс `Scanner`
* Переменные
* Арифметические операции
* Работа с процентами

---

## 🔍 Логика решения

Пока сумма на счёте меньше миллиона, каждый год начисляется 5% дохода.

Для этого используется цикл:

```java
while (balance < 1000000)
```

Внутри цикла:

```java
balance = balance + balance * 0.05;
```

или более короткая запись:

```java
balance *= 1.05;
```

После каждого начисления процентов увеличивается счётчик лет:

```java
years++;
```

---

## 📈 Формула роста вклада

Каждый год сумма увеличивается на 5%:

```text
Новая сумма = Текущая сумма + 5%
```

Например:

```text
5000 → 5250 → 5512.5 → ...
```

Рост происходит постепенно, пока баланс не достигнет миллиона рублей.

---

## 💡 Что понял

Эта задача помогла разобраться:

* когда удобно использовать цикл `while`;
* как моделировать процессы, которые продолжаются до выполнения условия;
* как считать сложный процент;
* как получать данные от пользователя через `Scanner`.

---

## 🚀 Возможные улучшения

* Сделать процентную ставку вводимой с клавиатуры.
* Позволить пользователю задавать желаемую сумму накоплений.
* Выводить баланс после каждого года.
* Рассчитать итоговую сумму, а не только количество лет.

---

## ✅ Итог

Получилась программа, которая рассчитывает, сколько лет потребуется для достижения финансовой цели при ежегодном начислении процентов.

Задача помогла закрепить работу с циклом `while`, вводом данных через `Scanner` и расчётом процентов.

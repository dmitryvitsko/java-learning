# 🐸 Помогаем Жабе перейти дорогу

## 📋 Задача
Программа помогает Жабе принять решение можно ли переходить дорогу
на основе правил дорожного движения.

---

## 🧠 Теория

### Булевы переменные для сложных условий
Сохраняем промежуточные результаты в булевы переменные
и используем их в финальном условии:
```java
boolean isOnCrosswalk = onCrosswalk.equals("да");
boolean isTrafficLightGreen = trafficLightGreen.equals("да");

// Финальное условие читается как обычная фраза:
if ((isOnCrosswalk && isTrafficLightGreen) ||
    (!isOnCrosswalk && !isVehicleApproaching && isSafePlace))
```

### Зависимые условия
Не все вопросы нужно задавать - одни зависят от других:
```java
if (isOnCrosswalk) {
    // спрашиваем про светофор
} else {
    // спрашиваем про машины
}

// вопрос про опасное место только если нет машин:
if (!isOnCrosswalk && !isVehicleApproaching) {
    // спрашиваем про поворот/перекресток/остановку
}
```

### scanner.nextLine() vs scanner.nextInt()
- `nextLine()` - читает всю строку включая Enter
- `nextInt()` - читает число, оставляет Enter в буфере
- Смешивать их опасно - может возникнуть `NoSuchElementException`

---

## 🛠 Что использовал
- Булевы переменные для хранения ответов пользователя
- Зависимые условия - не все вопросы задаются всегда
- `switch` для проверки типа опасного места
- Цикл `for` для подсказок по полосам

---

## 💡 Вывод
Сложную логику удобно разбивать на булевы переменные с говорящими именами.
Финальное условие из таких переменных читается почти как обычный текст.

# 🏋️ Силовая тренировка в зале

## 📋 Задача

Разработать программу круговой тренировки.

Условия:

* Всего 5 кругов.
* На нечётных кругах выполняется 3 отжимания.
* На чётных кругах выполняется 5 приседаний.
* После каждого круга нужно 7 раз покачать пресс.
* После завершения каждого круга выводится мотивирующее сообщение.

---

## 🧠 Что использовал

* Вложенные циклы `for`
* Условный оператор `if / else`
* Оператор остатка от деления `%`

---

## 🔍 Логика решения

Для определения типа круга используется оператор остатка от деления:

```java
if (i % 2 == 0)
```

Если остаток равен `0`, круг считается чётным и выполняются приседания.

Если остаток не равен `0`, круг считается нечётным и выполняются отжимания.

### Примеры

```java
4 % 2 = 0
5 % 2 = 1
```

---

## 🔄 Вложенные циклы

Внешний цикл отвечает за количество кругов:

```java
for (int i = 1; i <= 5; i++)
```

Внутренние циклы отвечают за количество повторений упражнений:

```java
for (int j = 1; j <= 3; j++)
```

```java
for (int j = 1; j <= 5; j++)
```

```java
for (int j = 1; j <= 7; j++)
```

---

## 💡 Что понял

Эта задача помогла лучше понять:

* как работают вложенные циклы;
* как использовать условия внутри циклов;
* как определять чётные и нечётные числа через оператор `%`;
* как комбинировать несколько изученных тем в одной программе.

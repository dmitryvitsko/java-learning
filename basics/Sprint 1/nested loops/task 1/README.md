# 📦 Курьер разносит посылки

## 📋 Задача
Курьер обходит все квартиры в доме - 10 этажей по 5 квартир.
Программа печатает этаж, номер квартиры и статус доставки.

---

## 🧠 Теория

### Вложенные циклы
Цикл внутри цикла - внешний идет по этажам, внутренний по квартирам на этаже.

\```java
for (int i = 1; i <= этажи; i++) {       // внешний - этажи
    for (int j = 1; j <= квартиры; j++) { // внутренний - квартиры
        // выполнится этажи * квартиры раз
    }
}
\```

Итого итераций: 10 * 5 = 50 - ровно столько квартир в доме.

### Формула номера квартиры
\```java
int currentFlat = (i - 1) * flatsNumber + j;
\```
- `i - 1` - сколько этажей уже пройдено
- `* flatsNumber` - квартир на каждом этаже
- `+ j` - текущая квартира на этаже

Например этаж 3, квартира 2: (3-1) * 5 + 2 = 12

---

## 🛠 Что использовал
- Вложенные циклы `for`
- Формула вычисления номера квартиры

---

## 💡 Вывод
Вложенные циклы удобны когда нужно перебрать двумерные данные -
этажи и квартиры, строки и столбцы, игровое поле.

# ✈️ Планировщик авиамаршрута

## 📋 Задача
Приложение помогает выбрать маршрут Москва-Париж с учетом
сезона, стоимости билетов и наличия британской визы.

---

## 🧠 Теория

### Правильная валидация ввода
```java
// Неправильно - пропускает отрицательные числа:
if (!(monthNumber > 12))

// Правильно - проверяем обе границы:
if (monthNumber >= 1 && monthNumber <= 12)
```

### Логика OR при определении сезона
```java
if (monthNumber <= 2 || monthNumber == 12) {
    season = "Зима"; // декабрь, январь, февраль
}
```
Декабрь - особый случай, не попадает в диапазон до 2,
поэтому добавляем его отдельно через `||`.

### Правильный порядок условий
```java
// Сначала самое выгодное условие:
if (ticketMoscowParis < ticketMoscowLondonParis) {
    // прямой дешевле
} else if (britainVisa == 1) {
    // через Лондон если есть виза
} else {
    // прямой как запасной вариант
}
```
Порядок условий важен - проверяем от самого приоритетного к запасному.

---

## 🛠 Что использовал
- Цикл `while` с валидацией ввода
- Оператор `||` для объединения условий
- Вложенные условия для логики маршрута

# 💰 Финансовое приложение v2

## 📋 Задача
Улучшенная версия финансового приложения с цифровым меню,
бесконечным циклом и возможностью выхода по команде 0.

---

## 🧠 Теория

### Бесконечный цикл while
\```java
while (true) {
    // выполняется бесконечно
}
// или
int command = -1;
while (command != 0) {
    // выполняется пока команда не 0
}
\```
Используется когда не знаешь заранее сколько раз пользователь
будет взаимодействовать с программой.

### break - выход из цикла
\```java
while (command != 0) {
    command = scanner.nextInt();
    if (command == 0) {
        break; // немедленный выход из цикла
    }
}
\```
`break` останавливает цикл в любой момент - не дожидаясь
проверки условия.

### Цифровое меню vs текстовые команды
Числа удобнее строк для меню:
- не нужно следить за регистром
- меньше ошибок при вводе
- `scanner.nextInt()` проще чем `scanner.next().equals()`

---

## 🛠 Что использовал
- Бесконечный цикл `while`
- `break` для выхода из цикла
- Цифровое меню вместо текстовых команд
- `scanner.nextInt()` для считывания команд

---

## 💡 Вывод
Бесконечный цикл + break - стандартный паттерн для интерактивных
консольных приложений. Пользователь сам решает когда выйти.

# 📷 Вклад Алисы на объектив

## 📋 Задача
Рассчитать сумму вклада через 3 года с учетом процентов,
ежемесячного пополнения и комиссии за обслуживание.

---

## 🧠 Теория

### Порядок операций в цикле - важен!
Неправильный порядок дает неверный результат:
```java
// Неправильно:
roubles -= monthMaintenance;
roubles += depositInterest;
roubles += monthRefill;

// Правильно - сначала проценты, потом комиссия, потом пополнение:
roubles += depositInterest;
roubles -= monthMaintenance;
roubles += monthRefill;
```

### Фиксированная комиссия от начальной суммы
```java
// Считается один раз ДО цикла:
double monthMaintenance = 0.001 * roubles; // 0.1% от 5000 = 5 руб/месяц
```
Комиссия не меняется каждый месяц - она фиксирована от первоначальной суммы.

### Месячная процентная ставка
```java
double monthRate = yearRate / 12; // 4% / 12 = 0.333% в месяц
double depositInterest = roubles * (monthRate / 100);
```

---

## 🛠 Что использовал
- Цикл `for` на 36 итераций
- Правильный порядок операций внутри цикла
- Фиксированная комиссия от начальной суммы

---

## 💡 Вывод
Порядок операций в финансовых расчетах критически важен.
Небольшая перестановка строк дает разный результат на выходе.

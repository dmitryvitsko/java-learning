# 💰 Финансовое приложение v6

## 📋 Задача
Добавлен поиск максимальной суммы расходов за неделю.

---

## 🧠 Теория

### Поиск максимума в массиве
```java
double maxExpense = 0; // стартовое значение
for (int i = 0; i < expenses.length; i++) {
    if (expenses[i] > maxExpense) {
        maxExpense = expenses[i]; // обновляем если нашли больше
    }
}
```
Классический алгоритм поиска максимума:
1. Берем стартовое значение 0
2. Идем по всему массиву
3. Если текущий элемент больше максимума - обновляем максимум
4. После цикла в maxExpense - самое большое значение

---

## 🛠 Что использовал
- Переменная-аккумулятор `maxExpense`
- Цикл `for` с условием обновления максимума

---

## 💡 Вывод
Поиск максимума - базовый алгоритм который встречается везде.
Та же логика используется для поиска минимума, только знак меняется на `<`
и стартовое значение берется максимально большим.

# 💰 Финансовое приложение v3

## 📋 Задача
Добавлена команда для записи ежедневных трат в массив.
Траты за один день суммируются при повторном вводе.

---

## 🧠 Теория

### Накопление значений в массиве
```java
expenses[day - 1] = expenses[day - 1] + expense;
```
Не перезаписываем старое значение, а добавляем к нему новое.
`day - 1` - потому что дни недели с 1, а индексы массива с 0.

### Смещение индекса
```java
// Пользователь вводит: 1 (понедельник)
// Индекс в массиве:    0
expenses[day - 1] // всегда вычитаем 1
```
Классическая ситуация когда нумерация для пользователя
и индексация массива отличаются.

---

## 🛠 Что использовал
- Массив `double[]` для хранения трат
- Накопление `expenses[i] = expenses[i] + expense`
- Смещение индекса `day - 1`
- Бесконечный цикл `while (true)` с меню

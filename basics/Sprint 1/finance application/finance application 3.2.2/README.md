# 💰 Финансовое приложение v5

## 📋 Задача
Добавлена команда для просмотра трат за всю неделю с разбивкой по дням.

---

## 🧠 Теория

### Смещение при выводе индекса
```java
// Индекс массива начинается с 0, но дни недели с 1
for (int i = 0; i < expenses.length; i++) {
    System.out.println("День " + (i + 1) + ". Потрачено " + expenses[i]);
}
```
`i + 1` в скобках - важно, иначе Java склеит число со строкой:
`"День " + i + 1` даст `"День 01"`, а не `"День 1"`.

---

## 🛠 Что использовал
- Цикл `for` для вывода всех элементов массива
- Смещение индекса `i + 1` для нумерации с 1

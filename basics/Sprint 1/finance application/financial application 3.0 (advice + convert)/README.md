# 💰 Финансовое приложение

## 📋 Задача
Объединение конвертера валют и советника по ужину в одно приложение с командами.

---

## 🧠 Теория

### Сравнение строк в Java
Строки нельзя сравнивать через `==` - только через `.equals()`:

```java
// Неправильно:
if (command == "convert")

// Правильно:
if (command.equals("convert"))
```
`==` сравнивает адреса в памяти, а `.equals()` сравнивает содержимое строки.

### Считывание строк через Scanner
- `scanner.next()` - считывает одно слово до пробела
- `scanner.nextLine()` - считывает всю строку целиком

---

## 🛠 Что использовал
- `scanner.next()` - считывание команды и кода валюты
- `command.equals()` - сравнение строк
- Вложенные `if/else` - логика команд и валют

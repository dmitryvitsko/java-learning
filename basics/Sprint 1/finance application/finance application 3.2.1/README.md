# 💰 Финансовое приложение v4

## 📋 Задача
При вводе траты баланс уменьшается на эту сумму.
Если баланс опускается ниже 1000 рублей - выводится предупреждение.

---

## 🧠 Теория

### Уменьшение переменной
```java
moneyBeforeSalary -= expense;
// то же самое что:
moneyBeforeSalary = moneyBeforeSalary - expense;
```
Оператор `-=` - сокращенная запись вычитания с присваиванием.

### Проверка порогового значения
```java
if (moneyBeforeSalary < 1000) {
    System.out.println("На вашем счету осталось совсем немного!");
}
```
Проверяем ПОСЛЕ изменения баланса - чтобы отреагировать
на актуальное значение.

---

## 🛠 Что использовал
- Оператор `-=` для уменьшения баланса
- Проверка порогового значения после каждой траты
- Вывод текущего баланса после операции

---

## 💡 Вывод
Финансовое приложение v4 - уже похоже на реальный банковский трекер.
Баланс обновляется в реальном времени, есть предупреждения.
Именно такую логику используют мобильные банки.

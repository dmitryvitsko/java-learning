# Советник по ужину

## Задача
Программа советует где поужинать в зависимости от остатка денег до зарплаты.

## Что использовал
- `Scanner` для считывания ввода
- Условия `if / else if / else`

## Что улучшил
Убрал лишние условия в else if - если дошли до ветки,
предыдущее условие уже гарантированно не выполнилось.

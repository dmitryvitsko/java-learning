# Корм для Байта

## Задача
Посчитать на сколько дней хватит 500 рублей на корм по 100 рублей каждые 3 дня.

## Что использовал
- Цикл `for`
- Условие остановки `i > 0`

## Что понял
Условие в цикле `for` можно писать так, чтобы не нужен был `break` -
цикл сам останавливается когда условие становится false.

# Калькулятор вклада

## Задача
Рассчитать итоговую сумму вклада с учетом сложного процента и бонуса каждые 3 года.

## Что использовал
- Цикл `for` для расчета по годам
- Условие `i % 3 == 0` для бонуса каждые 3 года
- Переменные `exchangeRate`, `interestRate`, `currencySymbol` для избежания дублирования

## Что улучшил
Перевод в рубли вынес в одно условие `if (currency != 1)` вместо трех блоков.

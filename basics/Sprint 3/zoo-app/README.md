# 🦁 Приложение для зоопарка

## 📋 Задача
Консольное приложение для управления списком животных зоопарка.
Добавление, удаление, поиск и очистка списка.

---

## ⚙️ Функциональность
- Показать всех животных в зоопарке
- Добавить животное в список
- Удалить животное из списка
- Очистить список
- Проверить есть ли животное в зоопарке

---

## 🧠 Теория

### ArrayList vs массив
```java
// Массив - фиксированный размер
String[] animals = new String[5];

// ArrayList - динамический, растёт сам
ArrayList<String> animals = new ArrayList<>();
```
ArrayList удобнее массива - не нужно знать размер заранее,
есть встроенные методы для всех операций.

### Основные методы ArrayList
```java
animals.add("Лев");          // добавить элемент
animals.remove("Лев");       // удалить элемент по значению
animals.clear();             // очистить список
animals.contains("Лев");     // проверить наличие → boolean
animals.isEmpty();           // проверить пустой ли список → boolean
animals.size();              // количество элементов
```

### Проверка перед удалением
```java
// Удалять только если список не пуст
if (!animals.isEmpty()) {
    animals.remove(animal);
}

// Очищать только если есть что очищать
if (!animals.isEmpty()) {
    animals.clear();
}
```
Защита от операций на пустом списке - хорошая практика.

### contains() вместо цикла
```java
// Раньше с массивом - нужен цикл
for (String a : animals) {
    if (a.equals(animal)) { ... }
}

// С ArrayList - одна строчка
boolean isAnimal = animals.contains(animal);
```

---

## 🛠 Что использовал
- `ArrayList<String>` - динамический список
- Методы: add(), remove(), clear(), contains(), isEmpty(), size()
- for-each цикл для вывода списка
- Консольное меню с командами

---

## 💡 Вывод
ArrayList решает главную проблему массивов - фиксированный размер.
Встроенные методы contains() и remove() убирают необходимость
писать циклы для базовых операций.

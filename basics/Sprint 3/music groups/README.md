# 🎸 Music Groups Directory

## 📌 Описание проекта

Консольное приложение на **Java** для работы с музыкальными группами с использованием коллекции **HashMap**.

Программа позволяет узнать год основания известных музыкальных групп, а также добавить в коллекцию свою любимую группу через удобное консольное меню.

Проект выполнен в рамках обучения на курсе **«Java-разработчик» от Яндекс Практикума** и посвящён изучению коллекции **HashMap**, работе с данными в формате «ключ — значение» и пользовательскому вводу.

---

## 🚀 Функциональность

- 🎵 Вывод списка музыкальных групп
- 🔍 Поиск года основания группы
- ➕ Добавление новой группы в коллекцию
- 📋 Консольное меню для взаимодействия с пользователем
- 💾 Хранение данных в `HashMap<String, Integer>`

---

## 🛠️ Используемые технологии

- Java
- HashMap
- Scanner
- Циклы (`while`, `for-each`)
- Условные конструкции (`if / else`)
- Коллекции Java

## 📖 Что нового изучил

Во время выполнения этого проекта познакомился с одной из самых популярных коллекций Java — **HashMap**.

Изучил:

- хранение данных по принципу **ключ → значение**;
- добавление элементов методом `put()`;
- получение значения по ключу через `get()`;
- перебор элементов с помощью `keySet()`;
- создание простого консольного интерфейса для работы с коллекцией.

Также закрепил работу со `Scanner` и ещё раз столкнулся с особенностями использования методов `nextInt()` и `nextLine()`.

## 🎯 Итог

В этом проекте впервые применил **HashMap** для хранения и быстрого поиска данных.

Теперь понимаю, почему эта коллекция так часто используется в реальных приложениях: она позволяет быстро находить нужную информацию по ключу, а код становится проще, понятнее и удобнее для дальнейшего развития.

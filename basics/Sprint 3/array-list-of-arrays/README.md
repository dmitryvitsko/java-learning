# 📋 Список массивов (ArrayList of arrays)

## 📋 Задача
Создать ArrayList который хранит массивы целых чисел.
Вывести каждый массив построчно через запятую.

---

## 🧠 Теория

### ArrayList может хранить массивы
```java
// ArrayList где каждый элемент - это массив int[]
ArrayList<int[]> list = new ArrayList<>();
list.add(new int[]{1, 2, 3});
list.add(new int[]{10, 20});
```
Тип элемента в угловых скобках - int[].
Каждый элемент списка это отдельный массив.

### Вывод массива через запятую без лишней запятой в конце
```java
// Первый элемент без запятой
System.out.print(arr[0]);
// Остальные с запятой перед ними
for (int i = 1; i < arr.length; i++) {
    System.out.print("," + arr[i]);
}
```
Стартуем с индекса 1 и добавляем запятую перед каждым следующим.
Так запятая не появляется в конце строки.

### Вложенный цикл - for-each + for
```java
for (int[] arr : list) {      // идём по списку
    for (int i = 1; i < arr.length; i++) { // идём по массиву
        System.out.print("," + arr[i]);
    }
}
```
Внешний цикл - по ArrayList.
Внутренний цикл - по элементам каждого массива.

---

## 🛠 Что использовал
- `ArrayList<int[]>` - список массивов
- for-each для перебора списка
- for для вывода элементов массива через запятую
- `System.out.print()` vs `System.out.println()`

---

## 💡 Вывод
ArrayList может хранить не только строки и числа но и массивы.
Трюк с запятой - стартовать вывод с индекса 1 чтобы
избежать лишней запятой в начале или конце.

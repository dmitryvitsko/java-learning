# 🎬 Самый кассовый фильм

## 📋 Задача
Метод находит самый кассовый фильм из трёх вариантов и возвращает его название.

---

## 🧠 Теория

### Метод с возвращаемым значением
```java
public static String findHighestGrossingFilm() {
    // логика
    return arrayNames[maxIndex]; // возвращает строку
}
```
Метод не void - он возвращает результат который можно использовать:
```java
String result = findHighestGrossingFilm();
```

### Поиск максимума с отслеживанием индекса
```java
int maxPrice = income1;
int maxIndex = 0;

for (int i = 0; i < arrayPrices.length; i++) {
    if (maxPrice <= arrayPrices[i]) {
        maxPrice = arrayPrices[i];
        maxIndex = i; // запоминаем индекс а не только значение
    }
}
return arrayNames[maxIndex]; // по индексу берем название
```
Храним не только максимальное значение но и его индекс -
чтобы потом найти соответствующее название фильма.

---

## 🛠 Что использовал
- Метод с возвращаемым типом `String`
- Два параллельных массива - цены и названия
- Поиск максимума с отслеживанием индекса

---

## 💡 Вывод
Параллельные массивы - когда данные связаны по индексу.
Индекс максимальной цены = индекс названия этого фильма.

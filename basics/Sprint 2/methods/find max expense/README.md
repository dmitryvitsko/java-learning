# 💸 Максимальная трата за неделю

## 📋 Задача
Метод принимает массив расходов за неделю и возвращает максимальное значение.

---

## 🧠 Теория

### Метод с параметром-массивом
```java
public static double findMaxExpense(double[] array) {
    // array - это ссылка на массив, не копия
}
```
Массив передается по ссылке - метод работает с оригинальным массивом.

### Double.MIN_VALUE как стартовое значение
```java
double max = Double.MIN_VALUE; // наименьшее возможное double
```
Вместо `0` берем минимально возможное значение типа double -
это гарантирует что любое число из массива окажется больше.
Если взять `0` - метод сломается на массиве из отрицательных чисел.

### Метод возвращает double
```java
public static double findMaxExpense(double[] array) {
    // ...
    return max;
}
```
Тип возвращаемого значения совпадает с типом элементов массива.

---

## 🛠 Что использовал
- Метод с параметром `double[]`
- `Double.MIN_VALUE` как безопасное стартовое значение
- Поиск максимума в цикле `for`

---

## 💡 Вывод
`Double.MIN_VALUE` - правильный старт для поиска максимума.
`0` как стартовое значение - частая ошибка которая ломает код
на отрицательных числах.

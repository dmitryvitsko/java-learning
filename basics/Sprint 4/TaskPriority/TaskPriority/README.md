# 📋 Task Priority Manager

Консольное Java-приложение для работы со списком задач и их приоритетами.

Проект выполнен в рамках обучения на курсе **«Java-разработчик»** от **Яндекс Практикума** и посвящён изучению **enum**, **коллекций ArrayList** и базовой фильтрации объектов.

---

## 📌 Описание проекта

Программа хранит список задач, каждая из которых имеет:

- приоритет (`HIGH`, `MEDIUM`, `LOW`);
- текстовое описание.

Основная задача программы — найти и вывести все задачи с **наивысшим приоритетом** (`HIGH`).

---

## ⚙️ Функциональность

- создание списка задач;
- хранение объектов в `ArrayList`;
- использование перечисления `enum` для приоритетов;
- фильтрация задач по приоритету;
- вывод наиболее важных задач на экран.

## 📚 Что было изучено

Во время выполнения задания закрепил следующие темы:

## 🎯 Чему научился

В этом проекте впервые использовал **enum** для моделирования фиксированного набора значений.

Такой подход позволяет:

- избежать строковых ошибок (`"HIGH"` vs `"HIGT"`),
- сделать код более безопасным,
- повысить читаемость программы.

Также закрепил работу с коллекциями объектов и научился фильтровать элементы по заданному условию.

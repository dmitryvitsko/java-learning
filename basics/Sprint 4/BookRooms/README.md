# 🏨 Royal Java Bed & Breakfast

Консольное приложение для управления бронированием номеров в мини-отеле **Royal Java Bed & Breakfast**.

Проект выполнен в рамках обучения на курсе **«Java-разработчик»** от **Яндекс Практикума** и посвящён изучению **инкапсуляции**, **модификаторов доступа** и взаимодействия между классами.

---

## 🚀 Возможности

- 🛏️ Забронировать номер
- 🚪 Освободить номер
- 📋 Проверить наличие свободных номеров
- 🔒 Скрыть внутреннее состояние объекта с помощью модификаторов доступа

## 🎯 Чему научился

В этом проекте впервые на практике применил инкапсуляцию.

Сделал внутреннее состояние объекта недоступным извне, скрыв поля `capacity` и `full`, а также вспомогательный метод `printRooms()`. Теперь взаимодействие с объектом происходит только через его публичные методы (`book()`, `free()`, `isFull()`), что делает код безопаснее, понятнее и соответствует принципам объектно-ориентированного программирования.

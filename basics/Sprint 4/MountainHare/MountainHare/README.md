# 🐇 Лес с зайцами — static поля и методы

## 📋 Задача
Создать симулятор леса с зайцами-беляками.
Цвет шерсти всех зайцев меняется одновременно в зависимости от сезона.

---

## 🧠 Теория

### static поле — одно на всех
```java
public static String color = "Серо-рыжий";
```
Обычное поле - у каждого объекта своё значение.
Static поле - одно значение для всех объектов класса.

Когда меняется сезон - меняется color у всех зайцев сразу.
Не нужно обходить каждого зайца и менять цвет вручную.
Это именно то что нужно - у всех зайцев цвет одинаковый.

### static метод
```java
public static void setSeason(String newSeason) {
    season = newSeason;
    if (newSeason.equals("Зима")) {
        MountainHare.color = "Белый";
    } else {
        MountainHare.color = "Серо-рыжий";
    }
}
```
Static метод не привязан к конкретному объекту.
Вызывается через имя класса: Forest.setSeason("Зима").
Внутри static метода можно работать только со static полями.

### Обращение к static полю через имя класса
```java
MountainHare.color = "Белый"; // правильно - через класс
hare.color = "Белый";         // работает но плохой стиль
```
Static поле принадлежит классу а не объекту.
Поэтому обращаться к нему лучше через имя класса.

### toString() со static полем
```java
@Override
public String toString() {
    return "Заяц-беляк: age=" + age +
           ", color=" + color; // color - static, общий для всех
}
```
toString() каждого зайца использует общий static color.
Когда color меняется - все toString() автоматически
вернут новое значение.

---

## 🛠 Что использовал
- `static` поле color - общее для всех объектов класса
- `static` метод setSeason() - изменяет общее состояние
- `@Override toString()` - текстовое представление зайца
- `ArrayList<MountainHare>` - список зайцев в лесу
- for-each для вывода всех зайцев

---

## 💡 Вывод
Static поля используют когда значение должно быть общим
для всех объектов класса. Классический пример - счётчик
объектов, константы, общее состояние.
Но злоупотреблять static не стоит - это усложняет тестирование
и может привести к неожиданным побочным эффектам.

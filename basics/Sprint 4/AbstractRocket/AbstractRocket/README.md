# 🚀 AbstractRocket

Учебный проект по Java, посвящённый изучению **абстрактных классов**.

## 📌 Описание

В этом задании реализована модель ракеты с использованием абстрактного класса.

В проекте изучается один из фундаментальных принципов объектно-ориентированного программирования — **абстракция**. Общие свойства и поведение ракеты вынесены в абстрактный класс, а конкретные типы ракет реализуют собственную логику через наследование.

## ⚙️ Что реализовано

- создание абстрактного класса;
- объявление абстрактных методов;
- наследование от абстрактного класса;
- переопределение методов в дочерних классах;
- использование полиморфизма при работе с объектами.

## 🛠️ Используемые технологии

- Java
- ООП
- Абстрактные классы
- Наследование
- Полиморфизм

## 🎯 Чему я научился

Во время выполнения задания я разобрался:

- чем абстрактный класс отличается от обычного;
- когда стоит использовать абстрактные методы;
- как вынести общую логику в базовый класс;
- как дочерние классы реализуют собственное поведение;
- почему абстракция помогает писать более понятный и расширяемый код.
